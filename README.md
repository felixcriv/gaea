# gaea

HTML parser for earthquake events data from funvisis.gob.ve

#Simple usage:

npm install

node server.js

Go to localhost:5000/events/5?l=en-US

#i18n support:

Use 'l' param query for event i18n support. Italian (it) and US English (en-US) supported. Default en-US.
